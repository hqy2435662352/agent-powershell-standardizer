# agent-powershell-standardizer
A high-reliability PowerShell skill for AI agents (Trae/Cursor/Windsurf). Eliminates Linux/Bash thinking in PowerShell scripts by enforcing Verb-Noun commands, object-oriented pipelines, path safety with Join-Path, explicit encoding, and -WhatIf support. Includes 15+ anti-pattern comparisons and best practices for enterprise-level script quality.
